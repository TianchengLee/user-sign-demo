模块:

  * [x] 登录

1. 点击登录按钮, 进行表单校验
2. 将用户输入的信息提交给服务器
3. 服务器返回token和当前用户信息, 将其存储到localStorage中
4. 提醒用户登录成功, 跳转到首页
